# health-care-providers
A Global System using Bootstrap, JavaScript, Laravel, and MySQL to register healthcare providers worldwide. The system allows users to search and connect with healthcare professionals, offering a seamless registration process and a secure, scalable platform for efficient provider management.

Staging Environment:
https://health-care-providers.vercel.app/

Tech Stacks:
Bootstrap, JS
Laravel Framework
MySQL DB

Primary Requirements:
Frontend Requirements:
Organization can register through standalone form.
Organizations and team members having bulk data can also upload a CSV or Excel file to upload their subsidiaries in bulk.

Backend Requirements:
Roles:
User or Organization.
Team Members.
Super Admin.
User Dashboard: This should be used by all the organizations registered using frontend form OR by the team members added from the backend by Super Admin.
User can add single organization listing and can manage its record.
User have the ability to add, edit or delete any record added by them.
User or organizations can see all the listing added by them.
Admin Dashboard: Below are some of the basic functions needed for the super admin. Layout for the admin to be used from the Health Monitor UI as explained in the initial video meeting:
User Management:
Can manage Organizations from the backend (Add, View, Edit, Delete, Block, Reset Password etc)
Can manage team members from the backend (Add, View, Edit, Delete, Block, Reset Password etc)
Listing Management:
Can manage all listings added by organizations and team members (Add, View, Edit, Delete, Block etc).
Search Function:
All users must have the ability to search a record within their account.
